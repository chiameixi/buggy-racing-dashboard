# overview
An interactive web-based dashboard for analyzing buggy racing performance through GPS telemetry visualization. This primarily targets buggy drivers, allowing them to review their racing lines, compare laps, and identify areas for improvement through visual overlays and time-based playback. This allows the GPX data collected in each practice session to be utilized more meaningfully.

demo video: https://drive.google.com/file/d/1g0ap0u6Aur5SGQ4g1Vlyv2f_3eEouRxl/view?usp=sharing 

# how to run
- npm install
- npm run dev and open up in browser
